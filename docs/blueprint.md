# **App Name**: Stacks DAO View

## Core Features:

- DAO Selection: Allows users to select a DAO either by inputting the contract address or choosing from a list of known DAOs on the Stacks blockchain.
- Treasury Overview: Provides a snapshot of the DAO's treasury, displaying key information such as the DAO name, STX balance, and last updated block height.
- Proposals Viewer: Lists all proposals with essential details including the Proposal ID, title/short description, and status (Active/Passed/Rejected).
- Proposal Detail Page: Offers an in-depth view of each proposal, including proposal metadata, voting results (yes/no), and creation block height.

## Style Guidelines:

- Background color: Dark charcoal gray (#333333) for a Bitcoin-inspired dark mode aesthetic.
- Primary color: Electric orange (#FF7800) to reflect Bitcoin branding. Avoid teal, as the user did not explicitly request it.
- Accent color: Light gray (#CCCCCC) for subtle highlights and readability.
- Font: 'Inter', a sans-serif font, suitable for headlines and body text
- Use simple, readable icons to represent different data points and proposal statuses.
- Prioritize clear information hierarchy with minimal distractions, focusing on readability and ease of use.
- Subtle transitions and animations to enhance user experience without being distracting.
